# Product Context

## Product Overview
Jinja2 template engine port for Dart 3. Provides server-side template rendering capabilities with variables, expressions, control structures, and template inheritance.

## Key Features
- Template rendering with variables and expressions
- Control structures (if, for, with, set, etc.)
- Template inheritance and includes
- Macros and imports
- Debug support with breakpoints
- Async rendering support
- Comprehensive error handling

## Target Users
- Dart developers needing template rendering
- Server-side Dart applications
- Web applications using Dart
